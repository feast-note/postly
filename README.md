## Infinite Canvas, Infinite Ideas (draft)

### [기술스택]
- **Next.js 16 (App Router)**
- **Tailwind CSS**
- **Sanity, Next-auth**
    
### 특징 

- **무한 캔버스 기반의 메모 관리 인터페이스 구현**
- **Headless CMS 기반의 데이터 아키텍처 설계**
    - **Sanity**를 활용하여 메모 데이터를 구조화하고, **Next-auth** 기반의 보안 인증 시스템을 구축

### 📝 완료한 작업

- 무한캔버스 구현: 무한 캔버스 공간에서 포스트잇을 자유롭게 배치 가능 
- 드래그&드랍: 개별 포스트잇을 원하는 위치에 배치할 수 있음
- 포스트잇수정: 포스트잇의 너비,좌표,색상,내용등을 수정할 수 있음
- 테스트모드: 로그인 없이 캔버스 테스트 이용 가능
