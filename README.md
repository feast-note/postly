## Infinite Canvas, Infinite Ideas (draft)

### [기술스택]
- **Next.js 14 (App Router)**
- **Tailwind CSS**
- **Sanity, Next-auth**
    
### 특징 

- **무한 캔버스 기반의 메모 관리 인터페이스 구현**
    - 라이브러리에 의존하지 않고 **Native DOM Event와 CSS Transform**을 활용하여 줌(Zoom) 및 드래그 인터랙션을 구현하며 사용자 경험(UX) 최적화
- **Headless CMS 기반의 데이터 아키텍처 설계**
    - **Sanity**를 활용하여 메모 데이터를 구조화하고, **Next-auth** 기반의 보안 인증 시스템을 구축하여 프론트엔드 외적 아키텍처 이해도 확장

### 📝 완료한 작업

- 무한캔버스 구현: 브라우저의 한계를 넘어서 사용자가 자유롭게 이동/확장하며 아이디어 배치할 수 있도록 구성
- 드래그&드랍: 개별 포스트잇 카드를 자유롭게 배치하고 이동시킬 수 있는 직관적인 사용자 인터랙션 구현

### 🚀 작업 예정 및 진행중

- **UX 최적화 (Z-Index Management)**: 카드 이동 시 활성화된 카드를 최상단으로 노출하여 시각적 혼동을 방지하는 동적 레이어 우선순위 로직 구현 예정
- **Grouping System**: 연관된 아이디어를 묶어 관리할 수 있는 그룹화 기능을 통해 대규모 아이디어 시각화 지원 예정
- 카드 추가 메뉴 구현 : 단순 텍스트를 넘어, 사용자의 목적에 맞는 다양한 형태(자유로운 크기 조절, 이미지 업로드 전용 카드, 순수 텍스트 카드 등)의 멀티미디어 카드 타입 확장 예정
